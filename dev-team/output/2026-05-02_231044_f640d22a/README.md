# User Story

> Write a Python CLI tool that reads a CSV file and generates a statistical summary report: column types, min/max/mean/median, missing values, and top-5 most frequent values per column

## Specification: CSV Statistical Summary CLI Tool

**Complexity:** medium

### Requirements

1. The CLI tool must accept a file path to a CSV file as input from the user.
2. The tool should detect and report the data type of each column (e.g., numeric, categorical, string).
3. For numeric columns, the tool must calculate and report minimum, maximum, mean, and median values.
4. For all columns, the tool should count and report the number of missing values.
5. The tool should identify and display the top 5 most frequent values for each column.
6. The tool must handle input validation, such as checking if the file exists and is a valid CSV file, and provide user-friendly error messages.
7. The tool should handle edge cases, such as columns with all missing values, non-numeric columns for statistical calculations, and empty CSV files gracefully.

### Acceptance Criteria

1. Given a valid CSV file with numeric and categorical data, when the tool is run, then it outputs column types, missing values, statistical measures for numeric columns, and top 5 frequent values per column.
2. Given a file path that does not exist or is not a CSV file, when the tool is run, then it displays an appropriate error message without crashing.
3. Given a CSV file where some columns have all missing values, when the tool is run, then it reports the missing values count and indicates no statistical summaries for those columns.
4. Given a CSV file with non-numeric columns, when the tool is run, then it does not attempt numeric statistical calculations on those columns and still reports missing values and top frequent values.
5. Given an empty CSV file, when the tool is run, then it informs the user that the file has no data to process.
6. Given any CSV input, when the tool is run, then it completes execution without unhandled exceptions and outputs the summary report.
