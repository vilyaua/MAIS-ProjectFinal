
# Demo Requests — OMKARA / КУБЕРРА

Тестовые user stories для AI-пайплайна в контексте импорта и логистики.

## 1. Калькулятор себестоимости SKU (simple)

Напиши Python-скрипт, который считает landed cost единицы товара на основе FOB-цены, морского фрахта, таможенной пошлины и НДС. На входе — JSON/CSV с позициями, на выходе — таблица с итоговой себестоимостью за штуку и за партию.

## 2. Container Loader CLI (simple)

Сделай CLI-утилиту, которая по списку коробок (длина × ширина × высота, вес, количество) считает заполнение 20ft / 40ft / 40HQ контейнера: процент использования объёма, общий вес, остаток места. С предупреждением о превышении весовых лимитов.

## 3. Invoice / Packing List Parser (medium)

Создай Python-библиотеку, которая парсит коммерческие инвойсы и упаковочные листы из Excel/PDF от китайских поставщиков и сверяет позиции с исходным заказом: совпадение артикулов, количеств, цен. Возвращает структурированный список расхождений.

## 4. Платёжная сверка по контрактам (medium)

Напиши Python CLI-инструмент: на вход — Excel с платежами поставщику и спецификациями по контракту (схема 30% / 70%), на выходе — отчёт о расхождениях: недоплаты, переплаты, нулевые балансы при реальных долгах, неверно посчитанные предоплаты. Поддержка валют USD/CNY/EUR.

## 5. REST API трекинга поставок (complex)

Построй REST API на Django/Flask для управления цепочкой поставок: SKU, поставщики, контракты, контейнеры со статусами (произведено → отгружено → в пути → на таможне → получено), привязка к инвойсам и платежам. SQLite/PostgreSQL, API-key аутентификация, CRUD по всем сущностям, фильтры по статусам и датам, экспорт отчётов в Excel.
