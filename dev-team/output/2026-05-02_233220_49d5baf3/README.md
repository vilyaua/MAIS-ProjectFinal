# User Story

> Write a Python CLI tool that reads a CSV file and generates a statistical summary report: column types, min/max/mean/median, missing values, and top-5 most frequent values per column

## Specification: CSV Statistical Summary CLI Tool

**Complexity:** medium

### Requirements

1. Implement a CLI tool that accepts a CSV file path as input argument.
2. Detect and report data types for each column in the CSV file (e.g., numeric, categorical, datetime).
3. Compute and display minimum, maximum, mean, and median values for numeric columns.
4. Identify and report the count of missing values for each column.
5. Identify and list the top-5 most frequent values for each column, regardless of type.
6. Include input validation to handle invalid file paths, unreadable files, or non-CSV formats and return meaningful error messages.
7. Ensure the tool handles large files efficiently and gracefully handles columns with all missing values or unique values.

### Acceptance Criteria

1. Given a valid CSV file, when the tool is executed, then it outputs column types correctly determined for each column.
2. Given numeric columns, when the tool processes the data, then the min, max, mean, and median values are computed and displayed correctly.
3. Given columns with missing values, when the tool processes data, then the count of missing values per column is reported accurately.
4. Given columns with categorical or any type of values, when the tool processes data, then top-5 most frequent values per column are listed correctly.
5. Given an invalid or missing input file path, when the tool is executed, then it exits gracefully with an error message indicating the problem.
6. Given columns with all missing values or unique values, when the tool processes data, then it does not crash and reports relevant statistics appropriately.
