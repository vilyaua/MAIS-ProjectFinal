### Requirements

2. The function should correctly convert the given temperature value from the source unit to the target unit using the appropriate formulas.
3. Input validation must be performed to ensure the temperature value is a valid number.
4. Input validation must ensure that the source and target units are among the supported units: Celsius, Fahrenheit, and Kelvin.
5. The function must handle edge cases such as extremely high or low temperatures within the valid physical range.

4. Given a temperature value below absolute zero in Kelvin, when the function is called, then it raises an exception or handles the input according to physical constraints.
5. Given conversions between all pairs of Celsius, Fahrenheit, and Kelvin, when the function is called, then it produces accurate results matching known conversion formulas.
