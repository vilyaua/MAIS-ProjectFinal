**Complexity:** simple

### Requirements

1. The script must accept text input from the user via the command line or standard input.
2. The script should generate ASCII art representation of the input text using a standard font style.
3. The script must validate that the input text is not empty and contains valid characters (letters, numbers, and common punctuation).
4. The script should handle errors gracefully, such as unsupported characters or input exceeding a reasonable length limit (e.g., 100 characters).

4. Given the user inputs text longer than 100 characters, when the script runs, then an error message prompts to shorten the input.
5. Given the user runs the script with a help flag, when the script runs, then usage instructions are displayed.
