# User Story

> Build a CLI password generator that supports custom length, uppercase, lowercase, digits, and special characters with a strength meter

## Specification: CLI Password Generator with Strength Meter

**Complexity:** medium

### Requirements

1. Implement a command-line interface (CLI) that accepts user inputs for password length and character set options including uppercase letters, lowercase letters, digits, and special characters.
2. Validate the password length input to ensure it is a positive integer within reasonable bounds (e.g., 8 to 128 characters).
3. Generate a random password based on the selected character sets and specified length.
4. Calculate and display a password strength meter based on criteria such as length, variety of character types included, and estimated entropy.
5. Handle input errors gracefully, providing user-friendly messages and guidance for valid inputs.
6. Allow users to request password generation multiple times without restarting the application.
7. Ensure the generated password always includes at least one character from each selected category to meet strength and complexity requirements.

### Acceptance Criteria

1. Given valid inputs for length and character sets, when the user requests password generation, then the CLI outputs a password matching the length and character set criteria.
2. Given an invalid length input (e.g., a non-integer or out of range), when the user inputs this value, then the CLI prompts an error message and requests a valid length.
3. Given user selection of multiple character sets, when a password is generated, then it includes at least one character from each chosen set.
4. Given a generated password, when the strength meter is displayed, then it accurately reflects the password complexity based on length and character variety.
5. Given repeated password generation requests in one session, when the user triggers a new generation, then a new unique password is output each time.
6. Given no character sets are selected, when the user attempts to generate a password, then the CLI prompts an error indicating that at least one character set must be selected.
