# User Story

> Сделай CLI-утилиту, которая по списку коробок (длина × ширина × высота, вес, количество) считает заполнение 20ft / 40ft / 40HQ контейнера: процент использования объёма, общий вес, остаток места. С предупреждением о превышении весовых лимитов.

## Specification: Container Load Calculator CLI

**Complexity:** medium

### Requirements

1. The CLI utility accepts a list of boxes with parameters: length, width, height (dimensions), weight, and quantity.
2. Calculate total volume of all boxes and compute percentage usage of each container type volume: 20ft, 40ft, 40HQ.
3. Calculate total weight of all boxes and compare it against the maximum weight capacity for each container type, triggering warnings if exceeded.
4. Calculate and display leftover capacity (volume and weight) for each container type after loading all boxes.
5. Input validation for box parameters: all dimensions, weight, and quantity must be positive numbers.
6. Error handling for invalid inputs (e.g., negative, zero or non-numeric values).
7. Handle edge cases such as empty lists and extremely large quantities that exceed container capacity.

### Acceptance Criteria

1. Given a valid list of boxes, when the utility runs, then it outputs volume utilization percentages for each container type.
2. Given box weights summing over the container weight limit, when the utility runs, then it outputs a weight limit exceeded warning for that container.
3. Given invalid box parameter inputs, when the utility runs, then it outputs appropriate error messages and does not crash.
4. Given an empty list of boxes, when the utility runs, then it outputs zero utilization and weight for all containers with no errors.
5. Given large quantities of boxes that exceed container volume, when the utility runs, then leftover volume is zero and warnings about overload are shown appropriately.
