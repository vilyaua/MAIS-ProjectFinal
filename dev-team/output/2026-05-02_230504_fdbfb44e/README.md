# User Story

> Create a Python library that converts Markdown text to HTML, supporting headers, bold, italic, links, code blocks, and lists

## Specification: Markdown to HTML Conversion Library

**Complexity:** medium

### Requirements

1. Support conversion of Markdown headers (#, ##, ###, etc.) to corresponding HTML header tags (h1, h2, h3, etc).
2. Support conversion of bold text marked by ** or __ delimiters to <strong> HTML tags.
3. Support conversion of italic text marked by * or _ delimiters to <em> HTML tags.
4. Support conversion of Markdown links [text](url) to HTML <a> tags with href attribute.
5. Support conversion of code blocks denoted by triple backticks (```) into <pre><code> HTML blocks preserving formatting.
6. Support conversion of ordered and unordered lists from Markdown to <ol> and <ul> HTML tags with <li> items.
7. Validate input to be a string containing Markdown text and handle empty or null input gracefully with appropriate error handling.

### Acceptance Criteria

1. Given valid Markdown text with headers when processed then corresponding HTML header tags are present in output.
2. Given Markdown text with bold and italic formatting when processed then <strong> and <em> tags appear correctly in the HTML output respectively.
3. Given Markdown links when processed then output contains proper <a href="..."> tags linking correctly.
4. Given Markdown code blocks when processed then output includes <pre><code> blocks preserving formatting and indentation.
5. Given Markdown ordered and unordered lists when processed then output contains appropriate <ol> and <ul> tags with nested <li> elements.
6. Given empty string or non-string input when processed then function returns empty string or raises a clear, descriptive error respectively.

## How to Run

Implemented a dependency-free Markdown to HTML conversion library.
