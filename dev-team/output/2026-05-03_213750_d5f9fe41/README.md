# User Story

> Create a Python library that converts Markdown text to HTML, supporting headers, bold, italic, links, code blocks, and lists

## Specification: Markdown to HTML Converter Library

**Complexity:** medium

### Requirements

1. Support conversion of Markdown headers (e.g., #, ##, ###) to corresponding HTML header tags (<h1>, <h2>, <h3>).
2. Support conversion of bold text syntax (e.g., **bold**) to HTML <strong> tags.
3. Support conversion of italic text syntax (e.g., *italic*) to HTML <em> tags.
4. Support conversion of Markdown links ([text](url)) to HTML <a href="url">text</a> tags.
5. Support conversion of code blocks (fenced with ``` or indented) to HTML <pre><code> blocks.
6. Support conversion of ordered and unordered lists to HTML <ol> and <ul> with <li> items respectively.
7. Validate input to ensure only valid Markdown syntax is processed and handle unsupported or malformed Markdown gracefully by leaving it unchanged or escaping it.

### Acceptance Criteria

1. Given a Markdown header, when converted, then it matches the corresponding HTML header tag.
2. Given a bold Markdown text segment, when converted, then it outputs the correct <strong> HTML tags.
3. Given an italic Markdown text segment, when converted, then it outputs the correct <em> HTML tags.
4. Given a Markdown link, when converted, then it produces the correct <a href> HTML anchor tag.
5. Given code blocks in Markdown, when converted, then it correctly wraps the content inside <pre><code> HTML tags.
6. Given ordered and unordered Markdown lists, when converted, then they produce the corresponding HTML <ol> and <ul> structures with <li> elements.
7. Given malformed or unsupported Markdown input, when converted, then the output preserves input (or escapes it) without crashing.
