# User Story

> Write a Python script that generates ASCII art from text input

## Specification: ASCII Art Generator from Text Input

**Complexity:** simple

### Requirements

1. The script must accept user input as a string of text.
2. The script must convert the input text into ASCII art representation using a standard font or style.
3. The script should validate that the input is non-empty and contains only printable characters.
4. If the input is invalid (empty or non-printable characters), the script should return an appropriate error message.
5. The script must output the ASCII art to the console or standard output.
6. The script should handle different input lengths gracefully, including long texts by wrapping or truncating if necessary.

### Acceptance Criteria

1. Given a valid text input, when the script is run, then it outputs the correct ASCII art representation matching the input text.
2. Given an empty string as input, when the script is run, then it returns an error message indicating input cannot be empty.
3. Given input with non-printable characters, when the script is run, then it returns an error message indicating invalid characters.
4. Given a long text input, when the script is run, then it outputs the ASCII art appropriately either wrapped or truncated to avoid console overflow.
5. Given a text input, when the script is run, then the output ASCII art only contains ASCII characters representing the input text.
