# User Story

> Build a CLI password generator that supports custom length, uppercase, lowercase, digits, and special characters with a strength meter

## Specification: CLI Password Generator with Strength Meter

**Complexity:** medium

### Requirements

1. The CLI tool shall allow users to specify the desired password length as a positive integer greater than zero.
2. The CLI tool shall provide options to include uppercase letters, lowercase letters, digits, and special characters in the generated password.
3. The tool shall validate user inputs to ensure at least one character type is selected and that the length is within a reasonable range (e.g., 1-128).
4. The tool shall generate a random password based on the selected options and specified length.
5. The CLI shall display a password strength meter evaluating the generated password's strength based on length and character variety.
6. The tool shall handle invalid inputs gracefully by displaying clear error messages and prompting for correct inputs.
7. The CLI application shall output the generated password to the user after generation along with the strength score or rating.

### Acceptance Criteria

1. Given valid inputs for length and character types, when the user requests a password, then a password matching the criteria is generated and displayed.
2. Given a length input less than or equal to zero, when the user attempts to generate a password, then the tool returns an error prompting for a valid positive length.
3. Given no character types are selected, when the user attempts to generate a password, then the tool returns an error instructing to select at least one character type.
4. Given a generated password, when the strength meter is calculated, then the displayed strength rating accurately reflects the password’s length and character diversity.
5. Given a password generation request with special characters included, when the password is generated, then the output contains at least one special character.
6. Given a password generation request with digits included, when the password is generated, then the output contains at least one digit.
