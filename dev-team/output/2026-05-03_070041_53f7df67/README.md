### Requirements

2. The script must convert the input text into ASCII art using a standard font style.
3. The script must display the generated ASCII art output in the console.
4. The script must validate that the input is not empty and contains only printable characters.
5. If the input is invalid (empty or non-printable characters), the script must display an appropriate error message.
