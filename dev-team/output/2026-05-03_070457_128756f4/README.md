# User Story

> Create a Python library that converts Markdown text to HTML, supporting headers, bold, italic, links, code blocks, and lists

## Specification: Markdown to HTML Converter Library

**Complexity:** medium

### Requirements

1. Provide a function to convert Markdown text input into HTML output.
2. Support the conversion of headers (levels 1 to 6) from Markdown to corresponding HTML header tags.
3. Support bold and italic text formatting in Markdown and convert them to HTML (<strong> for bold, <em> for italic).
4. Convert Markdown links ([text](url)) into HTML anchor tags.
5. Convert fenced code blocks in Markdown into HTML <pre><code> blocks preserving formatting.
6. Support Markdown unordered and ordered lists and convert them to corresponding HTML <ul>/<ol> and <li> elements.
7. Validate input to ensure it is a string and handle empty or null input gracefully by returning an empty string or appropriate message.

### Acceptance Criteria

1. Given valid Markdown input with headers, when the function is called, then the output must contain correct HTML header tags corresponding to Markdown syntax.
2. Given Markdown text with bold and italic syntax, when converted, then the output must include <strong> and <em> tags respectively.
3. Given a Markdown link syntax [text](url), when converted, then the output must contain an <a> tag with the correct href attribute and link text.
4. Given a fenced code block in Markdown, when converted, then the output must contain <pre><code> elements preserving the code content.
5. Given Markdown lists (both ordered and unordered), when converted, then output must include corresponding <ul>/<ol> and <li> tags in valid HTML format.
6. Given empty or non-string input, when conversion is requested, then output must return an empty string or a suitable error message without crashing.
7. Given complex input combining multiple supported features, when converted, then output must render all features correctly in the resulting HTML.
