# User Story

> Build a CLI password generator that supports custom length, uppercase, lowercase, digits, and special characters with a strength meter

## Specification: CLI Password Generator with Strength Meter

**Complexity:** medium

### Requirements

1. The CLI tool shall allow the user to specify the desired password length as a positive integer.
2. The tool shall allow the user to include or exclude character types: uppercase letters, lowercase letters, digits, and special characters.
3. The generated password must contain at least one character from each selected character type.
4. The tool shall evaluate the strength of the generated password based on length and character type diversity and display a strength meter (e.g., weak, medium, strong).
5. The tool shall validate user inputs and provide error messages for invalid inputs such as non-integer length or no character types selected.
6. The CLI shall output the generated password and its strength meter clearly to the user.
7. The tool shall handle edge cases such as minimum length less than the number of selected types by prompting the user accordingly.

### Acceptance Criteria

1. Given the user inputs a valid length and selects all character types, when generating the password, then the password contains uppercase, lowercase, digits, and special characters.
2. Given the user inputs a length less than the number of selected character types, when generating the password, then the tool prompts an error indicating length is too short.
3. Given the user inputs a non-integer or negative length, when attempting to generate a password, then the tool displays an appropriate input validation error message.
4. Given the user selects no character types, when attempting to generate a password, then the tool displays an error indicating at least one character type must be selected.
5. Given a generated password, when the strength is evaluated, then the strength meter displays the correct strength level based on predefined criteria.
6. Given the password generation is successful, when the result is displayed, then the CLI outputs the password and strength meter information clearly.
