# Demo Requests

Sample user stories for testing the AI Dev Team pipeline.

## 1. ASCII Art Generator (simple)

```
Write a Python script that generates ASCII art from text input
```

## 2. Password Generator CLI (simple)

```
Build a CLI password generator that supports custom length, uppercase, lowercase, digits, and special characters with a strength meter
```

## 3. Markdown to HTML Converter (medium)

```
Create a Python library that converts Markdown text to HTML, supporting headers, bold, italic, links, code blocks, and lists
```

## 4. CSV Data Analyzer (medium)

```
Write a Python CLI tool that reads a CSV file and generates a statistical summary report: column types, min/max/mean/median, missing values, and top-5 most frequent values per column
```

## 5. REST API with SQLite (complex)

```
Build a REST API with Flask for a task manager with SQLite persistence, user authentication via API keys, and CRUD operations for tasks with priorities and due dates
```
