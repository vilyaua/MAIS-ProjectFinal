# User Story

> Write a Python script that generates ASCII art from text input

## Specification: ASCII Art Generator from Text Input

**Complexity:** medium

### Requirements

1. The script must accept text input from the user through the command line.
2. The script must convert the input text into ASCII art representation.
3. The script must support multiple font styles for ASCII art generation.
4. The script must validate that the input text is not empty and contains only valid characters (e.g., alphanumeric and common punctuation).
5. The script must handle errors gracefully, including invalid input or unsupported characters.
6. The script must allow the user to choose the font style or use a default font if none is specified.
7. The output ASCII art must be displayed in the terminal/console for the user to see.

### Acceptance Criteria

1. Given the user provides valid text input, when the script runs, then ASCII art representation of the text is displayed in the terminal.
2. Given the user provides empty input, when the script runs, then an error message is displayed requesting valid input.
3. Given the user provides input with unsupported characters, when the script runs, then an error message is displayed specifying invalid characters.
4. Given the user does not specify a font style, when the script runs, then the script uses a default ASCII art font for output.
5. Given the user specifies a valid font style from the supported list, when the script runs, then the ASCII art output uses the chosen font style.
6. Given the script encounters an unexpected error during execution, when the script runs, then the script displays a user-friendly error message without crashing.
