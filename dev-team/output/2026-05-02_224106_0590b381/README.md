# User Story

> Write a Python script that generates ASCII art from text input

## Specification: ASCII Art Generator from Text Input

**Complexity:** medium

### Requirements

1. The script shall accept text input from the user via command line or function parameter.
2. The script shall convert the input text into ASCII art using standard ASCII characters.
3. The script shall support multi-line text input and handle it appropriately.
4. The script shall validate that the input text is not empty and consists of printable characters.
5. The script shall provide an option to output the ASCII art to the console or save it to a text file.
6. The script shall handle errors gracefully, such as invalid input or file write errors, providing meaningful messages to the user.
7. The script shall support different font styles or layouts for the ASCII art as optional settings.

### Acceptance Criteria

1. Given a valid single-line text input, when the script runs, then it should output the corresponding ASCII art to the console.
2. Given a multi-line text input, when the script runs, then it should generate correctly formatted ASCII art representing each line.
3. Given an empty or non-printable input, when the script runs, then it should show an error message and not generate ASCII art.
4. Given the output option set to save to file, when the script runs, then it should create a text file with the ASCII art content without errors.
5. Given a selection of different font styles, when the script runs, then the ASCII art output should reflect the chosen style appropriately.
