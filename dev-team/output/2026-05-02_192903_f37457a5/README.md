# User Story

> Create a Python library that converts Markdown text to HTML, supporting headers, bold, italic, links, code blocks, and lists

## Specification: Markdown to HTML Converter Library

**Complexity:** medium

### Requirements

1. Implement a function that accepts a Markdown string as input and returns the corresponding HTML string output.
2. Support conversion for the following Markdown elements: headers (levels 1-6), bold text, italic text, hyperlinks, inline code blocks, and unordered and ordered lists.
3. Ensure proper parsing and nesting of Markdown syntax to produce valid HTML elements.
4. Perform input validation to handle empty or null inputs gracefully by returning an empty string or appropriate message.
5. Handle edge cases such as malformed Markdown syntax without crashing, possibly by leaving unrecognized syntax as-is in output.
6. Provide comprehensive error handling to manage unexpected input types or encoding issues.
7. Design the library for easy integration and use as a standalone Python package.

### Acceptance Criteria

1. Given a Markdown string containing headers of various levels, when converted, then the output HTML contains corresponding <h1> to <h6> tags.
2. Given Markdown text with bold (**text**) and italic (*text*) syntax, when converted, then the output HTML contains <strong> and <em> tags respectively.
3. Given Markdown text with hyperlinks ([text](url)), when converted, then the output HTML contains <a href="url">text</a> elements.
4. Given Markdown text with inline code blocks (`code`), when converted, then the output HTML contains <code>code</code> elements.
5. Given Markdown text with ordered and unordered lists, when converted, then the output HTML contains <ol> or <ul> with nested <li> items.
6. Given an empty string input, when converted, then the output is also an empty string without errors.
7. Given malformed Markdown syntax that does not conform to supported patterns, when converted, then the library outputs the text without conversion but without raising exceptions.
