# User Story

> Напиши Python-скрипт, который считает landed cost единицы товара на основе FOB-цены, морского фрахта, таможенной пошлины и НДС. На входе — JSON/CSV с позициями, на выходе — таблица с итоговой себестоимостью за штуку и за партию.

## Specification: Landed Cost Calculator for Product Units

**Complexity:** medium

### Requirements

1. Accept input in JSON or CSV format containing product positions with fields including FOB price, sea freight, customs duty, and VAT.
2. Calculate the landed cost per unit by summing FOB price, allocated sea freight per unit, customs duty, and VAT for each product.
3. Calculate the total landed cost for the entire batch based on quantity for each product position.
4. Output the results in a tabular format showing: product identifier, landed cost per unit, total landed cost per batch.
5. Validate all monetary inputs to ensure they are non-negative numbers.
6. Handle cases of missing or malformed input data gracefully with error messages.
7. Support reading from and writing to files or standard input/output streams.

### Acceptance Criteria

1. Given a valid JSON or CSV input with product positions, when the script is run, then it outputs a table with correct landed cost per unit and total landed cost per batch for each product position.
2. Given input with negative values for prices or duties, when the script is run, then it raises an appropriate error indicating invalid input.
3. Given input missing any required field (FOB price, freight, customs duty, VAT), when the script is run, then it raises an error indicating missing data.
4. Given input with zero quantity, when the script is run, then the total cost for that batch is zero but the cost per unit is calculated correctly.
5. Given malformed JSON or CSV input, when the script is run, then it produces an error message indicating invalid format.
